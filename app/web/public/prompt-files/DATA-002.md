# Prompt: Informe de Insights Accionables

**Objetivo:** Generar un informe de insights de mercado con estructura Dato → Insight → Implicación → Acción, basado en datos del consumidor y análisis competitivo.

## Contexto para el Usuario
Usa este prompt cuando necesites fundamentar una estrategia con investigación de mercado, entender mejor al público objetivo o identificar oportunidades competitivas.

---

## Prompt Operativo

"Actúa como el **InsightsStrategist ('Telescopio')** de Live Developer. Necesito un informe de insights para [MARCA/PRODUCTO] dirigido a [PÚBLICO OBJETIVO] en [MERCADO/REGIÓN].

Genera el informe incluyendo:
1. **Contexto de Mercado:** Tamaño estimado, tendencias relevantes y dinámica competitiva.
2. **Perfil del Consumidor:** Buyer persona data-driven con demografía, psicografía, motivaciones y barreras.
3. **5 Insights Clave:** Cada uno con la estructura: **Dato** (evidencia) → **Insight** (verdad humana) → **Implicación** (qué significa para la marca) → **Acción** (qué hacer).
4. **Análisis Competitivo:** 3-5 competidores con fortalezas, debilidades y diferenciadores.
5. **Oportunidades Identificadas:** 3 oportunidades de mercado no explotadas.
6. **Recomendaciones Estratégicas:** 3-5 acciones priorizadas por impacto y viabilidad."

---

## Guía de Uso
- **Input necesario:** Marca/producto, público objetivo, mercado geográfico, datos existentes si los hay.
- **Output esperado:** Informe ejecutivo listo para presentar a cliente o alimentar estrategia interna.

## Ejemplo de Uso
**Input ejemplo:** "App de educación financiera para millennials ecuatorianos. No tenemos datos propios aún."

**Resultado esperado:**
- contexto del mercado fintech ecuatoriano,
- buyer persona millennial,
- 5 insights clave con estructura,
- análisis de competidores,
- recomendaciones priorizadas.
