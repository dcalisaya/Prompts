# Prompt: Customer Journey Map

**Objetivo:** Crear un mapa del journey del cliente completo con touchpoints, emociones, pain points y oportunidades de mejora por fase.

## Contexto para el Usuario
Usa este prompt cuando necesites entender la experiencia completa del cliente con una marca, identificar fricciones y diseñar mejoras de UX/servicio.

---

## Prompt Operativo

"Actúa como el **CXDesigner ('Empatía')** de Live Developer. Necesito un Customer Journey Map para [TIPO DE CLIENTE] de [EMPRESA/MARCA] que [OBJETIVO DEL CLIENTE].

Genera el journey map completo incluyendo:
1. **Persona:** Perfil del usuario con nombre ficticio, contexto, motivaciones y frustraciones.
2. **Fases del Journey:** Awareness → Consideration → Purchase → Onboarding → Uso → Retention → Advocacy.
3. **Por cada fase:** Touchpoints (puntos de contacto), acciones del cliente, pensamientos/emociones (escala 😡→😊), pain points y oportunidades.
4. **Momentos de Verdad:** 3-5 momentos críticos donde la experiencia se gana o se pierde.
5. **Gap Analysis:** Brechas entre la experiencia actual y la experiencia ideal.
6. **Acciones de Mejora:** 5-10 mejoras priorizadas por impacto en satisfacción y esfuerzo de implementación (matriz 2×2).
7. **Métricas por Fase:** Qué medir en cada etapa (ej: NPS post-compra, CES en onboarding, churn en retención)."

---

## Guía de Uso
- **Input necesario:** Empresa/marca, tipo de cliente, objetivo del cliente, canales de contacto existentes.
- **Output esperado:** Journey map narrativo con diagnóstico y plan de acción.

## Ejemplo de Uso
**Input ejemplo:** "App de banca digital para millennials ecuatorianos. El journey desde descarga hasta primera transacción."

**Resultado esperado:**
- persona del millennial bancario,
- journey de 7 fases,
- emociones por touchpoint,
- momentos de verdad,
- 7 mejoras priorizadas.
