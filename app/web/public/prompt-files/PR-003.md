# Prompt: Brief de Media Training

**Objetivo:** Preparar a un vocero para entrevistas con medios mediante la definición de mensajes clave, Q&A anticipado, bridging techniques y simulación de preguntas difíciles.

## Contexto para el Usuario
Usa este prompt antes de cualquier aparición en medios (TV, radio, prensa, podcasts) para que el vocero tenga mensajes alineados, respuestas preparadas y técnicas de manejo de entrevista.

---

## Prompt Operativo

"Actúa como el **PRStrategist ('Vocero')** de Live Developer. Necesito preparar un brief de media training para [NOMBRE DEL VOCERO], [CARGO], de [EMPRESA], que tendrá una entrevista en [MEDIO/FORMATO] sobre [TEMA].

Genera el brief completo incluyendo:
1. **Mensajes Clave (Key Messages):** 3 mensajes principales que el vocero debe comunicar sin importar la pregunta.
2. **Puntos de Soporte:** 2-3 datos, cifras o ejemplos que respaldan cada mensaje clave.
3. **Q&A Anticipado:** 10-15 preguntas probables con respuestas sugeridas (incluyendo 3-5 preguntas hostiles).
4. **Técnicas de Bridging:** Frases puente para redirigir preguntas incómodas hacia los mensajes clave.
5. **Errores a Evitar:** Lista de 5 trampas comunes (especulación, datos no confirmados, lenguaje técnico excesivo).
6. **Tips de Formato:** Recomendaciones específicas según el medio (TV: vestimenta y postura; radio: ritmo de voz; prensa: citas quotables)."

---

## Guía de Uso
- **Input necesario:** Nombre y cargo del vocero, medio y formato, tema principal, contexto sensible si aplica.
- **Output esperado:** Documento de preparación listo para sesión de entrenamiento.

## Ejemplo de Uso
**Input ejemplo:** "Juan Pérez, Director General de EcoTech, entrevista en Teleamazonas (TV en vivo) sobre innovación tecnológica en Ecuador."

**Resultado esperado:**
- 3 mensajes clave con puntos de soporte,
- Q&A con preguntas hostiles,
- técnicas de bridging,
- tips específicos para TV.
