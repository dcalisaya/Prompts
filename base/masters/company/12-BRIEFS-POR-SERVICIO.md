# Briefs Estandar por Servicio

Este documento define el minimo de informacion que cada area debe pedir antes de producir, cotizar o ejecutar.

## Regla transversal

Todo brief debe responder al menos:

- que quiere lograr el cliente,
- que pieza o sistema necesita,
- para cuando lo necesita,
- quien aprueba,
- que activos ya existen,
- que restricciones hay.

## Brief comercial general

- nombre del cliente
- contacto responsable
- necesidad principal
- servicios solicitados
- fecha objetivo
- presupuesto referencial si existe
- canal de entrega esperado

## Brief para audiovisual

- objetivo del video o pieza
- tipo de pieza
- cantidad de piezas
- duracion estimada
- formato requerido
- locacion
- fechas tentativas
- voceros o talento
- referencias visuales
- branding disponible

## Brief para podcast

- nombre del podcast
- tematica
- publico objetivo
- cantidad de episodios
- duracion por episodio
- invitados o formato
- CTA principal
- plataformas destino

## Brief para marketing y funnel

- producto o servicio
- propuesta de valor
- buyer persona
- nivel de consciencia
- objetivo de conversion
- ticket o valor de referencia
- activos existentes
- canales prioritarios

## Brief para pauta

- objetivo de campana
- plataforma
- presupuesto de medios
- duracion
- audiencias
- geografia
- activos creativos disponibles
- pixel o tracking disponible
- conversion esperada

## Brief para branding y diseno

- objetivo de la pieza
- formato
- mensajes clave
- branding existente
- referencias
- medidas o specs
- versionado requerido

## Brief para web y desarrollo

- problema a resolver
- usuarios objetivo
- funcionalidades requeridas
- integraciones
- stack deseado si aplica
- accesos existentes
- dependencias externas
- criterio de exito

## Brief para IA y automatizacion

- proceso actual
- tarea a automatizar
- entradas
- salidas
- herramientas involucradas
- frecuencia
- riesgo operativo
- validacion humana requerida

## Regla de operacion

Si el brief no esta completo, el equipo puede emitir recomendacion preliminar o proforma preliminar, pero no debe cerrar alcance final.
