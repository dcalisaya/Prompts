# Prompt: Generador de Arquitectura de Funnel

**Objetivo:** Crear un mapa estructural completo de un embudo de ventas, definiendo cada etapa desde la captación hasta la conversión final.

## Contexto para el Usuario
Usa este prompt cuando necesites definir el "esqueleto" de una campaña. Ideal para integrarse con la infraestructura tecnológica de Live Developer.

---

## Prompt Operativo

"Actúa como el **Estratega Digital ('Brain')** de Live Developer. Mi objetivo es vender [PRODUCTO/SERVICIO] a un avatar de [BUYER PERSONA]. 

Diseña una arquitectura de funnel completa que incluya:
1. **Etapa TOFU (Atracción):** Define el Lead Magnet y las fuentes de tráfico recomendadas.
2. **Etapa MOFU (Nutrición):** Estructura de la Landing Page (secciones clave) y secuencia de emails (3-5 correos).
3. **Etapa BOFU (Conversión):** Oferta irresistible y Call to Action final.
4. **Stack Tecnológico:** Especifica qué herramientas de Live Developer (CRM, Hosting, Analytics) son necesarias para que este funnel funcione."

---

## Guía de Uso
- **Input necesario:** Nombre del producto, precio sugerido y público objetivo.
- **Output esperado:** Documento estratégico con el flujo paso a paso y copy sugerido para los titulares principales.

## Ejemplo de Uso
**Input ejemplo:** producto: "Mentoria de ventas B2B", precio sugerido: "USD 800", publico objetivo: "duenos de pymes de servicios".

**Resultado esperado:**
- lead magnet,
- estructura TOFU / MOFU / BOFU,
- secciones de landing,
- secuencia de correos,
- stack tecnico recomendado.
