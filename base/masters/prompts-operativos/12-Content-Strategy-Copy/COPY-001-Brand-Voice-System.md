# Prompt: Brand Voice System

**Objetivo:** Diseñar un sistema de tono de voz y lineamientos de estilo que permita a una marca comunicar de manera consistente en web, redes, email, PR y ventas.

## Contexto para el Usuario
Usa este prompt cuando una marca suene inconsistente, genérica o demasiado dependiente del estilo de cada persona que redacta.

---

## Prompt Operativo

"Actúa como el **ContentStrategist ('Editor')** de Live Developer. Necesito construir el sistema de voz de marca para [EMPRESA], que opera en [INDUSTRIA] y quiere sonar [ATRIBUTOS DESEADOS].

Genera:
1. **Diagnóstico de Voz Actual:** Cómo suena hoy la marca y qué incoherencias presenta.
2. **Principios de Voz:** 4 a 6 atributos centrales con explicación práctica.
3. **Do / Don't:** Qué sí debe decir la marca y qué debe evitar.
4. **Jerarquía de Mensajes:** Mensaje principal, mensajes de soporte y claims permitidos.
5. **Léxico de Marca:** Palabras preferidas, palabras a evitar y expresiones distintivas.
6. **Adaptación por Canal:** Cómo se expresa en web, redes, email, ventas y PR.
7. **Ejemplos Aplicados:** Antes/después de copy para 3 canales.
8. **Checklist de Validación:** Cómo revisar si una pieza respeta la voz."

---

## Guía de Uso
- **Input necesario:** Industria, atributos deseados, audiencia, canales y ejemplos de comunicación actual.
- **Output esperado:** Documento de brand voice con reglas claras y ejemplos aplicados.

## Ejemplo de Uso
**Input ejemplo:** "Firma boutique de consultoría financiera. Quieren sonar experta, clara y menos acartonada."

**Resultado esperado:**
- principios de voz,
- do/don't,
- léxico sugerido,
- adaptación por canal,
- ejemplos antes/después.
