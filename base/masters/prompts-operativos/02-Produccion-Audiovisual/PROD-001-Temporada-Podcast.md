# Prompt: Estructurador de Temporada de Podcast

**Objetivo:** Diseñar el arco narrativo y la lista de temas para una temporada completa de podcast (10-20 episodios).

## Contexto para el Usuario
Usa este prompt para el caso de estudio de los "20 Podcasts". Asegura que el contenido sea relevante y retenga a la audiencia.

---

## Prompt Operativo

"Actúa como el **Director Audiovisual ('Soul')**. Necesito estructurar una temporada de [NÚMERO] episodios para un podcast llamado [NOMBRE DEL PODCAST], enfocado en [TEMÁTICA].

Genera una matriz que incluya:
1. **Título del Episodio:** Atractivo y optimizado para YouTube/Spotify.
2. **Gancho (Hook):** La pregunta o afirmación central para iniciar el video.
3. **Puntos Clave:** 3 sub-temas a tratar en cada episodio.
4. **CTA sugerido:** Qué queremos que haga el oyente en este episodio específico."

---

## Guía de Uso
- **Input necesario:** Concepto general y cantidad de episodios.
- **Output esperado:** Tabla organizada con la planificación de toda la temporada.

## Ejemplo de Uso
**Input ejemplo:** "Quiero una temporada de 12 episodios para un podcast de liderazgo y ventas."

**Resultado esperado:**
- tabla de episodios,
- hook por episodio,
- puntos clave,
- CTA por capitulo,
- coherencia de temporada.
