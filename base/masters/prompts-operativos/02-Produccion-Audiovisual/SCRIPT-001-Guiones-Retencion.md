# Prompt: Generador de Guiones Hook-Retención-CTA

**Objetivo:** Producir guiones rápidos para videos verticales o introducciones de podcasts que maximicen el tiempo de visualización.

## Contexto para el Usuario
Ideal para Reels, TikToks y Shorts que el equipo de Live Developer debe producir.

---

## Prompt Operativo

"Actúa como el **Director Audiovisual ('Soul')** especialista en retención. Escribe un guion para un video de [DURACIÓN] sobre [TEMA].

Estructura el guion de la siguiente manera:
1. **El Hook (0-3 seg):** Un gancho visual o auditivo disruptivo.
2. **El Valor (Cuerpo):** Desarrollo rápido con frases cortas y dinámicas. Evita el relleno.
3. **La Retención:** Incluye un 'Open Loop' a la mitad para mantener el interés.
4. **El CTA (Cierre):** Un llamado a la acción claro y de beneficio directo."

---

## Guía de Uso
- **Input necesario:** Tema central y plataforma (ej. TikTok vs LinkedIn).
- **Output esperado:** Guion listo para locución con sugerencias de texto en pantalla.

## Ejemplo de Uso
**Input ejemplo:** "Escribe un guion de 30 segundos para TikTok sobre errores comunes al vender servicios."

**Resultado esperado:**
- hook inicial,
- desarrollo rapido,
- open loop,
- CTA final,
- texto sugerido en pantalla.
