# Masters Layer

## Estructura canonica

- `catalog/`: catalogo oficial de servicios.
- `company/`: navegacion, operacion, pricing, reglas y capa empresarial.
- `agents/`: prompts maestros por especialidad.
- `prompts-operativos/`: prompts listos para tareas reales.
