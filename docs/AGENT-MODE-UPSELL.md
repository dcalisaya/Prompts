# Agent Mode: Upsell

## Proposito

Este modo se usa cuando el usuario quiere detectar continuidad, expansión o siguiente fase a partir de un trabajo ya existente.

## Cuándo activarlo

Señales típicas:

- "qué sigue"
- "continuidad"
- "siguiente fase"
- "ampliar"
- "postventa"

## Objetivo del modo

Detectar una ruta de continuidad coherente con el caso actual sin forzar una venta artificial.

## Qué debe detectar el agente

- servicio o proyecto ya existente
- oportunidad lógica de expansión
- valor real de esa siguiente fase
- mejor momento para proponerla

## Preguntas mínimas recomendadas

- ¿Qué entregable o fase ya se completó?
- ¿Qué resultado quiere extender el cliente?
- ¿Qué necesidad nueva o derivada apareció?

- `projects/` para revisar continuidad en casos reales

## Formato de salida esperado

- oportunidad detectada
- servicio o fase sugerida
- argumento operativo o comercial
- siguiente acción sugerida

## Errores a evitar

- forzar upsell sin base
- sugerir expansión sin conexión con el caso real
- sonar genérico o vendedor sin criterio

## Definición de terminado

El modo está bien ejecutado cuando la propuesta de continuidad se siente natural, defendible y útil.
